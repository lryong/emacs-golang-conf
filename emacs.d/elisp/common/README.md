# 关于common #
common目录主要用于存放一些公用的插件。

## exec-path-from-shell ##
[exec-path-from-shell](https://github.com/purcell/exec-path-from-shell)主要是为了解决在Mac OS X系统下通过GUI启动的emacs无法读取shell配置的环境变量问题。
